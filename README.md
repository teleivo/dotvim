# Installation

## Clone dotfiles repo

```bash
git clone https://github.com/teleivo/dotvim.git ~/.vim
```

## Install Vundle

```bash
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

## Install vim plugins

```bash
vim +PluginInstall +qall
```

## Create symlink to vimrc

```bash
ln -s ~/.vim/vimrc ~/.vimrc
```
